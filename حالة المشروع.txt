نبذة:
منصة ويب تعليمية لتحليل الأسهم الأمريكية (أداة فحص — لا توصيات).
مبنية بـ Python Flask، بيانات حقيقية من 3 مصادر.


───────────────────────────────────────────────
■ ما تم إنجازه (مكتمل وشغّال ✅)
───────────────────────────────────────────────

[1] قاعدة البيانات (models.py)
    3 جداول: watchlist, stock_cache, signals.

[2] جلب البيانات:
    - FMP        → بيانات مالية + سعر + تاريخ أسعار (لحساب ATR)
    - Finnhub    → سعر لحظي (مصدر تأكيد ثانٍ يرفع الثقة)
    - SEC EDGAR  → معاملات المطلعين (بدون مفتاح)

[3] الحسابات (scoring.py):
    - Piotroski Score (9 نقاط) — محسوب يدوياً ومتحقّق
    - Catalyst Score (0-100) — أوزان افتراضية (موضّحة في الواجهة)
    - خطة ATR تعليمية (دخول / وقف / هدف)

[4] الصفحات (شغّالة):
    - الرئيسية: ماسح + فلاتر (Piotroski، Catalyst، السعر، القطاع)
    - تقرير السهم: سعر + مقاييس + Piotroski + Catalyst + ATR + مطلعين
    - قائمة المتابعة: إضافة/حذف + تتبّع العائد
    - المقارنة: عدة أسهم جنباً لجنب

[5] الثيم الداكن الاحترافي (مستوحى من التصميم المرجعي).

المبادئ المطبّقة: None ≠ 0 ، Unit Guards ، لا توصيات ، لا أرقام ملفّقة.


───────────────────────────────────────────────

───────────────────────────────────────────────
■ الخطوة المتبقية الوحيدة: النشر على Railway
───────────────────────────────────────────────

الهدف: الحصول على رابط عام يفتحه أي أحد.

الخطوات عند العودة:
  1. railway.app → لوحة التحكم → New Project
  2. Deploy from GitHub repo → اختر stock-analysis-platform
  3. أضف قاعدة بيانات: + New → Database → PostgreSQL
  4. أضف متغيرات البيئة (Variables) للخدمة:
       FMP_API_KEY       = (المفتاح من ملف .env)
       FINNHUB_API_KEY   = (المفتاح من ملف .env)
       (DATABASE_URL يُضاف تلقائياً مع PostgreSQL)
  5. Settings → Networking → Generate Domain (للرابط العام)

ملاحظة: حساب Railway تجريبي برصيد منخفض — قد يطلب ترقية.


───────────────────────────────────────────────
■ تشغيل الموقع محلياً على جهازك
───────────────────────────────────────────────

  1. افتح الترمنال داخل مجلد المشروع
  2. اكتب:  py app.py
  3. افتح المتصفح على:  http://127.0.0.1:5000


───────────────────────────────────────────────
■ هيكل الملفات
───────────────────────────────────────────────

  app.py                  ← التطبيق الرئيسي (المسارات)
  models.py               ← جداول قاعدة البيانات
  requirements.txt        ← المكتبات المطلوبة
  Procfile                ← أمر التشغيل على Railway
  .env                    ← المفاتيح السرّية (لا يُرفع)
  .gitignore              ← يحمي .env من الرفع
  services/
      fmp_client.py       ← بيانات FMP
      finnhub_client.py   ← سعر Finnhub
      edgar_client.py     ← معاملات المطلعين (SEC)
      scoring.py          ← Piotroski + Catalyst + ATR
      analysis.py         ← تجميع تقرير السهم
      screener.py         ← الماسح + التخزين المؤقت
  templates/              ← صفحات الواجهة (HTML)
  static/style.css        ← الثيم الداكن
